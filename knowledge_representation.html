TEI/XML document

Some exerpts of the Wikipedia article on Chiune Sugihara has been encoded as a TEI/XML document. Selected entities are annotated inline using elements such as <persName>, <placeName>, <orgName>, and <rs>. Each entity is described in the back matter using dedicated list elements (listPerson, listPlace, listOrg, listBibl, listEvent, listObject) and reconciled to authority files including Wikidata, VIAF, and OCLC. Quotation is also marked up using <quote>.

TEI → HTML (XSLT)

The TEI/XML document is transformed into a HTML page using a Python script built on the xml.etree.ElementTree library. The script reads the back matter to build an index of entities, then processes the body text, wrapping each annotated entity in a colour-coded span linked to its Wikidata entry. The output also includes an entity index table and a relations table derived from listRelation.

TEI → RDF

The TEI/XML document is transformed into an RDF dataset in Turtle format using a Python script built on the rdflib library. The script maps each entity to the class defined in the conceptual model (e.g. foaf:Person, schema:City, crm:E5_Event, skos:Concept) and each relation to the corresponding property (e.g. schema:workLocation, crm:P15_was_influenced_by). Every entity is linked to its Wikidata counterpart via owl:sameAs. The dataset contains 45 triples in total.
